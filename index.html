Penetration Testing & Vulnerability Research Cheatsheet

⚠️ IMPORTANT — AUTHORIZED USE ONLY:
  • Only run these commands against systems you own or have explicit written authorization. Misuse is illegal.
  • This material is for education, professional training, and defensive hardening in controlled lab environments.

1. Recon & OSINT

Footprinting and information gathering.

whois example.com
dig +short example.com
theHarvester -d example.com -b all

2. Subdomain & DNS Enumeration

Discover subdomains and DNS records.

subfinder -d example.com
amass enum -d example.com
dnsenum example.com

3. Port Scanning & Service Enumeration

Identify open ports and running services.

nmap -sC -sV -oA scan_results example.com
masscan -p1-65535 example.com

4. Vulnerability Scanning

Automated vulnerability detection.

nmap --script vuln example.com
nikto -h example.com
wpscan --url example.com

5. Web Application Testing

Web security testing tools.

sqlmap -u https://example.com/page?id=1 --batch
gobuster dir -u https://example.com -w wordlist.txt
ffuf -u https://example.com/FUZZ -w wordlist.txt

6. Fuzzing

Automated input testing for vulnerabilities.

afl-fuzz -i input -o output ./target_binary
boofuzz -c config.py
wfuzz -c -w payloads.txt --hc 404 https://example.com/FUZZ

7. Exploit Development & PoC Testing

Safe PoC testing in lab environments.

msfconsole
python3 poc.py
./exploit_binary target_ip

8. Reverse Engineering & Binary Analysis

Analyze binaries safely.

gdb ./binary
radare2 ./binary
ghidra

9. Network & Protocol Analysis

Capture and analyze network traffic.

tcpdump -i eth0 -w capture.pcap
wireshark capture.pcap
ettercap -T -q -M arp:remote /target_ip/ /gateway/

10. Password Attacks & Bruteforce

Testing password strength safely.

hydra -l admin -P wordlist.txt ssh://example.com
john --wordlist=passwords.txt hash.txt

11. Active Directory / Windows Testing

Windows domain security assessment.

crackmapexec smb example.com -u user -p pass
bloodhound
impacket-secretsdump

12. Linux & Privilege Escalation

Local Linux enumeration and privilege escalation.

linpeas.sh
sudo -l
ps aux | grep root

13. Social Engineering & Phishing Simulation

Phishing awareness & simulated attacks.

gophish
setoolkit
evilginx2

14. Reporting & Documentation

Record findings safely and professionally.

KeepLabNotes.md
Jira/Trello tickets for remediation
Generate PDF or HTML reports for clients

15. Safety & Lab Best Practices

Maintain safe, isolated, and ethical testing environments.

Use VM snapshots
Isolate lab network
Always have authorization
Document everything
