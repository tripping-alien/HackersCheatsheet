1) Recon & OSINT OSINT Recon

Footprinting, WHOIS, DNS lookups, and basic HTTP checks.

# Set target
TARGET=example.com

# WHOIS lookup
whois $TARGET

# DNS: A + MX + NS records
dig +noall +answer $TARGET A
dig +noall +answer $TARGET MX
dig +noall +answer $TARGET NS

# HTTP headers check
curl -I -L --max-redirs 5 https://$TARGET

# Certificate names (crt.sh)
curl -s "https://crt.sh/?q=%25.$TARGET&output=json" | jq -r '.[].name_value' | sort -u

Commands gather domain info, DNS records, HTTP headers, and passive certificate data.
